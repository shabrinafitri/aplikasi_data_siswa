Cara bikin Table :
- bikin database
- klik kanan di kolom Table
- klik 'Table...'
- lalu bikin table nya..
- kalo mau set primary key : klik kanan value yg mau di set, lalu klik 'Set Primary Key'
- Untuk execute pake shortcut CTRL + S
